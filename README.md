# AI Job Agent

An AI-powered automation system that:

- scans job portals
- matches jobs intelligently
- automates applications
- tailors resumes dynamically

## Tech Stack

- Python
- Playwright
- PostgreSQL
- Ollama
- FastAPI

## Current Progress

- [x] Playwright browser automation setup
- [x] LinkedIn job card extraction
- [x] Direct job URL extraction
- [x] Multi-page job detail scraping
- [x] Structured job data collection

## In Progress
